# PIP3D Engine

**PIP3D** — это экспериментальный 3D-движок, разработанный для работы на ограниченных устройствах вроде **ESP32**. Архитектура строится вокруг минимального набора базовых примитивов, простых нодовых структур и оптимизированных алгоритмов, позволяющих рендерить сцены в реальном времени даже при небольшом объёме памяти и низкой частоте **CPU**.

Движок поддерживает работу с полигонами, базовую трансформацию объектов, освещение, тени и коллизии. Реализованы собственные алгоритмы ускорения, ориентированные на низкое потребление ресурсов, а также модульность, позволяющая расширять систему дополнительными функциями и датчиками, если движок используется на физических платформах.
